TABELAS DADOS

-dCadastroPropriedadesCRM = base de cadastro de propriedades do CRM, é a principal base e tem conexão com o dCadastroPropriedadesERP

-dCadFilial = Base de Cadastro de Filial, Regional e UF

-dCadastroPropriedadesERP = base de cadastro de propriedades do ERP, e tem conexão com o dCadastroPropriedadesCRM

-fHistoricoVendas = Base histórica de movimentação dos clientes a nível e produto

-fAstec = Base de atendimentos de assitencia técnica dos consultores a cliente

-dCarteira = Base de cadastro da carteira de clientes ativos, onde há a informação da carteira e filial que o cliente está vinculado

-dSegmentacao = Base de cadastro que fornece a informação da segmentação do cliente, indicando em que categoria o mesmo se encontra. Essencial para dividir os clientes em grupos

-fEntregaGrãos = Base de dados das movimentações de entregas de grãos dos clientes

-fPlanoSafra = Base de dados que indica o potencial de plantio de cada produtor. Obs: nem todos os clientes possuem essa informação
